Embedding <exposure-triangle>

This page mimics a Quarto document. The widget below was inserted with nothing more than one script tag and one custom-element tag. Three.js loads automatically; the only asset it needs is scene.png sitting next to the script.

<script src="exposure-triangle.js"></script>
<exposure-triangle width="100%" height="560px"></exposure-triangle>

Options are plain attributes: accent, scene (photo URL), floor-grid="false", spin, width, height. You can also mount from JS with ExposureTriangle.mount('#id', { accent:'#2A6FDB', width:'100%', height:'600px' }), and listen for the change event to read the resolved ISO / shutter / aperture.
